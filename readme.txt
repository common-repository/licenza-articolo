== Description ==

Questo plugin aggiunge un piccolo box che indica per ogni articolo del blog con info sulla licenza dell'articolo. In questo box c'è scritto che il testo dell'articolo articolo è di proprietà dell'autore dell'articolo e rilasciato sotto licenza Creative Commons.

== Installation ==

L'installazioen è semplice

1. Download the plugin da wordpress.org 
1. Estrai i file 
1. Upload  nella cartella `/wp-content/plugins/` 
1. Attiva attraverso  'Plugins' menu in WordPress.
